Sprite Animator  создан для анимации любого набора изображений, в форматах jpg, png, gif. 

Подключение:
<script type="text/javascript" src="jquery-1.9.1.min.js"></script>  
<script type="text/javascript" src="spriteanimator.js"></script>

Пример работы:

<script type="text/javascript">
imageArray = [
  'images/image001.jpg',
  'images/image002.jpg',
  'images/image003.jpg',
  'images/image004.jpg',,
  'images/image005.jpg'
  ];
  
  $(document).ready(function(){
       spriteanimator.init(imageArray);
  });
</script> 

Управление:

Для удобства просмотра слева сверху, можно установить ширину и высоту экрана с кадрами.

Кадры переключаются кнопками < назад  , вперёд >.

Анимация включается по нажатию на кнопку “играть”, и выключается либо по нажатию на ту же кнопку(надпись на ней меняется на “пауза”), либо по нажатию на кнопку “стоп”.

Если нажать на “пауза”, тогда воспроизведение начнётся с  текущего кадра.

Если нажать на “стоп”, воспроизведение начнётся сначала.

Галочка “повтор” зацикливает воспроизведение, и если анимация дошла до последнего кадра, она повторяется сначала.

Каждый кадр можно настроить  отдельно,  в списке под изображением.
