Национальные особенности русской философии

Ландшафт и природные явления (детская презентация)

Трёхмерная графика


Описание:

Скипт презентации сделан в виде клиент-серверного приложения.

Каждая презентация находится в своём каталоге в виде набора файлов-шаблонов.

1 слайд - это 2 файла: превью и код слайда + изображения слайда.

Файл-шаблон - это просто текстовый документ с html-кодом, в файле с расширением .slide

В корне каталога каждой презентации есть файл с именем manifest.xml, в нём содержатся список всех файлов презентации.
